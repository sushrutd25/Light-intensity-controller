# Light Intensity Control System Using Arduino Uno

## Project Overview

Developed an automated light intensity control system using Arduino Uno, LDR sensor, and PWM-based LED brightness control. The system adjusts LED brightness according to ambient light conditions, improving energy efficiency and user comfort.

## Objectives

* Automate light intensity control
* Implement PWM-based brightness adjustment
* Interface sensors with Arduino Uno
* Understand embedded control systems

## Components Used

* Arduino Uno
* LDR (Light Dependent Resistor)
* LED
* Resistors
* Breadboard
* Connecting Wires

## Working Principle

The LDR senses ambient light intensity and sends an analog signal to the Arduino Uno. The Arduino reads this value using its ADC (Analog-to-Digital Converter) and processes the data. Based on the measured light intensity, the Arduino generates a PWM (Pulse Width Modulation) signal to control the LED brightness.

When the surrounding environment becomes darker, the PWM duty cycle increases, resulting in higher LED brightness. When ambient light increases, the PWM duty cycle decreases, reducing LED brightness automatically.
